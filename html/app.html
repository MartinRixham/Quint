Quint

Instructions

Quint performs symbolic manipulation of tensors representing quantum systems. It uses the convention \(\ket{\theta} = \begin{pmatrix} \cos{\frac{\theta}{2}} \\ \sin{\frac{\theta}{2}} \end{pmatrix}\), so the computational basis is \[\ket{0} = \begin{pmatrix} 1 \\ 0 \end{pmatrix}, \ket{\pi} = \begin{pmatrix} 0 \\ 1 \end{pmatrix}\] State vectors are automatically normalised so \[\ket{0} + \ket{\pi} = \begin{pmatrix} \frac{1}{\sqrt{2}} \\ \frac{1}{\sqrt{2}} \end{pmatrix}\]
